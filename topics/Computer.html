Computer

What is a computer?

-Derived from the phrase "to compute," a computer is an electronic device that can store large amounts of data and perform various operations on the data. It is an electronic device that can store and work with large amounts of information.

Characteristics of Computer

  • Retrieving Data and Program
  • Automation
  • Versatility
  • Consistency
  • Communications
  • Speed
  • Accuracy
  • Reliability
  • Storage
  • Arithmetical and Logical Operations

Parts of a Computer

  • Hardware

    • -A hardware is any computer part that you can touch with your fingers.

    • Examples:

      • Computer Case
      • CPU
      • Monitor
      • Keyboard & Mouse
      • Disk Drive, Zip Drive, CD-ROM, DVD
      • Hard Drive
      • Speakers
      • Printer

    • Three Types of Hardware

      • Input Device

        • -It basically means getting data into the computer to be processed.

        • Examples:

          • Keyboard, Mouse
          • Trackball, Touch Pad
          • Light Pen, Laser Scanner
          • Pointing Stick
          • Touch Screen
          • Bar Code Reader, Scanner
          • Microphone, Joystick
      • Output Device

        • -Any information inputted or processed by the computer that will be displayed on an output device.

        • Examples:

          • Monitor
          • Printer
          • Speakers
          • Headphones
          • Modem
          • Fax
      • Storage Device

        • -It is a place to keep data that has been processed so that it can be retrieved at a later time to be used again.

        • Examples:

          • Hard Disk
          • Floppy Disk
          • CD's, DVD's
          • Magnetic Tape
          • Flash Memory, Jump Drive

    Devices that can either be input, output, or storage:

    • System unit

      • - The main case of a computer. It contains storage devices, the power supply, cooling fans, and the CPU.

    • Motherboard

      • - The Motherboard can either be an input or an output device.

      • Computer Chip

        • - Very small pieces of silicon or other semi-conducting material onto which integrated circuits are embedded.

      • Circuit Board

        • -A thin board containing computer chips and other electronic components.

      • Motherboard o System Board

        • -The main circuit board inside the system unit.

    • Central Processing Unit

      • -Circuitry and components packaged together and connected directly to the motherboard.

    • Memory

      • -Chip based storage.

      • RAM (Random Access Memory)

        • -Computer's main memory.

      • ROM (Read-only Memory)

        • -Non-volatile chips located on the motherboard into which data or programs have been permanently stored.

      • Flash Memory

        • -Type of non-volatile memory that can be erased and reprogrammed.

    • Power Supply Unit (PSU)

      • -Converts main AC to low-voltage regulated DC power for internal components of a computer.

    Other Components

    Heat is a continuing problem for CPU and computer manufacturers.

    • Fans

      • -Used on most personal computers.

    • Heat Sinks

      • -Small components typically made out of aluminum with fins that help to dissipate heat.

    • Water Cooling Systems

      • -Cool the computer with liquid-filled tubes.

  • Software

    • -It is the programs and applications that tell the computer what to do and how to look. Software refers to the programs that a computer executes to perform a function. It is also intangible and modifiable.

    • Two Types of Software

      • Operating System Software

        • -The system software that directly controls and manages the computer hardware and its basic system operations.

        • Examples:

          • DOS, Windows 95, 98, ME, NT, XP, Vista, Windows 7
          • Unix

            • -The oldest operating system that was also the first open source OS.It was designed as an interactive time-sharing systems at Bell Labs in 1969.

          • Linux

            • -A 32-bit multitasking operating software that is both command-driven and graphics-driven. It can run almost any type of application on almost any type of computer.

          • Mac OS

            • -Or the Macintosh Operating System, it comes with Apple Computer's Macintosh line of personal computers and workstations. it is developed by Apple Corp.

          • The rest of the MAC system (6, 7 ,8, 9, 10)
          • iOS

            • -Formerly known as iPhone OS, iOS is a mobile operating system that presently powers many of the company's mobile devices, including the iPhone, iPad, and iPod touch. iOS user interface is based upon direct manipulation, using multi-touch gestures.Interface control elements include sliders, switches, and buttons.

          • Android

            • -A mobile operating system developed by Google, based on the Linux kernel and designed primarily for touchscreen mobile devices such as smartphones and tablets.

          • Windows

            • -Designed in the mid-1980's and was Microsoft Corporation's answer to the MAC OS. It was designed as an operating environment which ran on top of DOS rather than as an operating system itself.

        • Categories of Operating Systems

          • User Interface

            • -Users communicate with computer software. It accepts input from the user in the form of commands and data input.

            • 2 Types of User Interface

              • Text User Interface

                • -Shows text, and its commands are usually written from the keyboard to the command line.

              • Graphical User Interface

                • -Carried out by clicking and moving buttons, icons, and menus with the mouse.

          • Command Line OS

            • -Command driven operating system enable users to communicate with the computer by inputting commands based on a standard syntax.

          • Graphics Based

            • -Uses the concept of WIMP(Windows, Icons, Menus, Pointers).

            • Examples:
              • Linux
              • Windows
              • Mac OS
      • Application Software

        • -Programs that work with operating system software to help the computer to do specific types of work. Application Software is specific to the task it is designed for and can be as simple as a calculator application or as complex as a word processing application.

        • 6 Types of Application Software

          • Business Software

            • -Includes word processors, spreadsheets, and database programs.

          • Communication Software

            • -Allows Computers to communicate with other computers.

          • Graphics Software

            • -Software that allows users to create and manipulate graphics.

          • Education and Reference Software

            • -Programs that help teach new material and ideas, and programs that can be used to find information.

          • Entertainment and Leisure Software

            • -As its name suggests, this type of software consists of programs for entertainment, in short, games.

          • Integrated Software

            • -Combines several types of software into one program or package.
